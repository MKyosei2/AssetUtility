# AssetUtility Portfolio Evidence

## 使い方

1. Unityで `Tools > Asset Utility Evidence > Create Evidence Pack` を実行します。
2. `Assets/AssetUtilityEvidence/Reports` のMarkdownを開きます。
3. `Assets/AssetUtilityEvidence/Screenshots` のBefore / After / Comparison PNGをREADMEやポートフォリオに貼ります。
4. ConsoleをClearしてから実行し、Errorが出ていない状態のスクショを残します。

## 出力される証拠

- Before / After の三角形数
- Texture Max Size の変更結果
- 穴あき検出用のBoundary Edge比較
- 非多様体・退化三角形・不正Indexチェック
- Before / After / Comparison PNG
- 検証Scene
