# Generated Validation Reports

This directory is the standard location for portfolio validation artifacts generated by CI or local dry-runs.

Expected generated files:

```text
portfolio_validation_report.md
portfolio_validation_report.json
assetutility_dry_run_sample_plan.json
```

The validation harness is dry-run by default. It does not modify Unity scenes, prefabs, textures, materials, or meshes. Errors are written into Markdown/JSON reports before the script exits with a non-zero code.

Local command:

```bash
python scripts/portfolio_validation.py --dry-run --report-dir Docs/Reports
```

The dry-run sample plan is not a replacement for Unity Editor validation. It exists to make the intended optimization, rollback, and report workflow inspectable in CI.
